# Dream-Game
My most ambitious project yet. 

# Plot Synopsis
Uventahl, United States of America; 2025. After losing his father to an overseas New York trip gone wrong, ... moves the state of Uventahl alongside his step father to finally let his fathers' soul rest once and for all while on a fight aganist the gods of time.

# Gameplay Synopsis 
This game is a hybrid between an action RPG and an open world sandbox adventure game. The game will be based on a fictionalised version of Utah and will feature multiple chapters. 

The game will most likely take about 40-50 hours to complete the main campaign, but 100% completion will take a minimum of 100 hours to complete. 

# Achievements
369 achievements will be to collect for the player which will range from simple tasks like reaching the next chapter to discovering the entire map.
